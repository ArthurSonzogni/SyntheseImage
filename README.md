SyntheseImage
=============

| | |
| ------------- | ------------- |
| What  | School project with advanced OpenGL  |
| By  |  @ArthurSonzogni and @Eusmilis  |
| When | Nov 2014 |



#### Features : 
- Phong shading
- Shadow

#### Advanced features :
- Defered shading
- Unlimited lights
- Bump mapping
- Ambient occusion (SSAO)
- Ambiant reflection (SSAR)
- Non Photorealistic Rendering (Pencil drawing)
