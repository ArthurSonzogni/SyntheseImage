SyntheseImage
=============

A school project with advanced OpenGL.
By @ArthurSonzogni and @Eusmilis

#### Features : 
- Phong shading
- Shadow

#### Advanced features :
- Defered shading
- Unlimited lights
- Bump mapping
- Ambient occusion (SSAO)
- Ambiant reflection (SSAR)
- Non Photorealistic Rendering (Pencil drawing)
